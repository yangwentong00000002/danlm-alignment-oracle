# DanLM Alignment Oracle

Temporary private repository used to export `danlm_alignment_samples.json` on a
GitHub Actions macOS runner.

Run the `Export DanLM Alignment Samples` workflow manually, download the
`danlm_alignment_samples` artifact, and copy the JSON back to:

```text
tools/danlm_export/work/danlm_alignment_samples.json
```
